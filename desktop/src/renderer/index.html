ℹ️ 使用说明
工作原理：
  1. 选择一个工作空间根目录
  2. 每个 DeepSeek 对话自动创建独立子目录
  3. 子目录以会话 ID 命名 (UUID)
  4. AI 可在该目录下读写文件、执行命令

元数据文件：
  • desktop.ini - Windows 文件夹名称/图标
  • .directory - Linux/Mac 文件夹信息
  • .deepseek-agent/ - 设置和日志
